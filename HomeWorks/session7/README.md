# Session 7: Linux Package Management

This session contains two separate homework tracks:

- `deb-homework.md` and `deb-answers.md` for Debian/Ubuntu-style systems
- `rpm-homework.md` and `rpm-answers.md` for RPM-based systems

Preparation scripts:

- `prepare-deb-lab.sh`
- `prepare-rpm-lab.sh`

Use the track that matches the package management family of the Linux system you are working on.
